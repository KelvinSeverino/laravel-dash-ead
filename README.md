# Laravel-Dash-EAD

## ❓ Para que serve?
Este repositorio se trata de um projeto de web desenvolvido em laravel 10 para aprendizado realizado pelo curso de Laravel Dashborad EAD da EspecializaTI.

## 💻 Pré-requisitos
Antes de começar, verifique se você atendeu aos seguintes requisitos:
* docker
* docker-compose
* composer

### 💻 Como executar

Baixar repositório
```sh
git clone https://github.com/KelvinSeverino/laravel-dash-ead.git
```

Acessar diretório do repositorio
```sh
cd laravel-dash-ead/
```
